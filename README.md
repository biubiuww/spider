## 目录

- **1024**： 数字社区的图片

- **baiduMap**： 简单调用百度地图的api完成区域类的信息检索，需要用到开发账号

- **cmanuf**：机械工业出版社的pdf下载？**烂尾，bug太多，不修了**

- **novel**：盗版小说的爬虫...存储到mongodb中

- **qicai**：七彩英语（英文原著）的PDF下载

- **umei**： 批量下载图片

- **video_grils**: 关键词：快手、随机、小姐姐、下载

- **yasee1**：年久失修代码，现在的网站已经修改了逻辑
